# Banking Information System 💳

## 📌 Project Overview
This project is a **Core Java Banking Information System** developed as part of a **4-week internship program**.  
It is a **console-based banking application** that simulates the basic functionalities of a real banking system.

The project was developed using **Core Java concepts** such as:
- Classes and Objects
- Constructors
- Methods
- ArrayList
- Conditional Statements
- Loops
- Package Handling
- Date and Time

The system allows users to register, log in securely, manage their account, perform transactions, and view account-related details.

## 👨‍💻 Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)
- Collections (ArrayList)
- Console-based User Interface
