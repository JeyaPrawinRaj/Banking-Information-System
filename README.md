# Banking Information System 💳

## 📌 Project Overview

This project is a **Core Java Banking Information System** developed as part of a 4-week internship program.
It simulates basic banking operations using a console-based application.

---

## 👨‍💻 Technologies Used

* Java (Core Java)
* OOP Concepts
* Collections (ArrayList)
* Console-based UI

---

## 📅 Week 1 - Basic Setup

* User Registration
* Account Creation
* Unique Account Number Generation
* Login System with Password Authentication

---

## 📅 Week 2 - Core Features

* Improved Login System
* Account Management
* Code Structure using Classes (User, Account, Transaction)
* Better Menu Navigation

---

## 📅 Week 3 - Banking Operations

* Deposit Functionality
* Withdraw Functionality
* Balance Checking
* Error Handling (Insufficient Balance, Invalid Input)
* Menu-driven Banking Interface

---

## ⚙️ Features Implemented

* User Registration & Login
* Account Management
* Deposit & Withdrawal
* Balance Inquiry
* Basic Error Handling

---

## 🚀 How to Run

```bash
javac BankingSystem/*.java
java BankingSystem.BankingSystem
```
